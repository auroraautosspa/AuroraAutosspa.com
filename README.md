A professional, fully-responsive website template for a car detailing business built with HTML, CSS, and JavaScript.

### 📱 Responsive Design
- Fully mobile-responsive layout
- Desktop, tablet, and mobile optimized
- Hamburger menu for mobile navigation

## 🎨 Customization

### Colors
The template uses CSS variables for easy customization. Edit the root variables in `styles.css`:

```css
:root {
    --primary-color: #1e40af;      /* Main blue */
    --secondary-color: #f97316;    /* Orange accent */
    --accent-color: #06b6d4;       /* Cyan accent */
    /* ... more variables */
}
```

### Content
All text content can be easily edited directly in `index.html`. Look for sections clearly marked with comments like:
```html
<!-- Home Section -->
<!-- About Section -->
<!-- Contact Section -->
etc.
```

### Images
Replace placeholder images:
- Hero background image
- About section image
- Gallery images
- Service icons (FontAwesome icons included)
